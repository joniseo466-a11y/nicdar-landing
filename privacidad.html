¿Qué es NicDar?

NicDar es una extensión de Chrome que proporciona herramientas de análisis de YouTube para ayudar a los usuarios a descubrir oportunidades de nicho en YouTube.

¿Qué datos recopilamos?

Cuando inicias sesión con Google, recopilamos:

  • Tu dirección de correo electrónico
  • Tu nombre

También generamos y almacenamos:

  • Un Device ID único y no identificable personalmente, almacenado localmente en tu navegador. Se utiliza exclusivamente para aplicar los límites del plan gratuito y prevenir el uso fraudulento desde el mismo dispositivo.

También recopilamos datos de uso anónimos como:

  • Número de scans realizados
  • Número de veces que usas Scans y Global Search, exclusivamente para controlar el uso de la API. NicDar no puede ver ni registra los canales o vídeos que visitas.

¿Cómo usamos tus datos?

  • Para autenticar tu cuenta y darte acceso a la extensión
  • Para monitorizar el uso y mejorar el servicio
  • Para aplicar los límites de uso según tu plan de suscripción

¿Dónde se almacenan tus datos?

Tus datos se almacenan en servidores seguros con cifrado en reposo y en tránsito, siguiendo las prácticas estándar del sector.

Servicios de terceros

NicDar utiliza los siguientes servicios de terceros:

  • Google OAuth — para el inicio de sesión
  • YouTube Data API v3 — para obtener datos públicos de vídeos y canales
  • Google Translate — para traducir resultados en el Global Search
  • Stripe — para el procesamiento de pagos. No almacenamos datos de tarjeta en nuestros servidores.

Compartición de datos

No vendemos, cedemos ni compartimos tus datos personales con terceros. Tus datos solo se utilizan para prestarte el servicio de NicDar.

Conservación de datos

Tus datos se conservan mientras tu cuenta esté activa. Puedes solicitar la eliminación de tus datos en cualquier momento contactándonos.

Seguridad

Implementamos medidas de seguridad técnicas y organizativas para proteger tus datos frente a accesos no autorizados, pérdida o alteración.

Tus derechos

  • Acceder a tus datos personales
  • Solicitar la eliminación de tus datos
  • Retirar tu consentimiento en cualquier momento desinstalando la extensión

YouTube API Services

NicDar utiliza los servicios de YouTube API. Al usar NicDar, aceptas quedar vinculado por la Política de Privacidad de Google. Puedes revocar el acceso desde tu página de permisos de Google.

Comunicaciones por email

Podemos enviarte actualizaciones del producto, consejos y noticias sobre NicDar ocasionalmente. Puedes darte de baja en cualquier momento haciendo clic en el enlace de cancelación incluido en cada email.

Cambios en esta política

Podemos actualizar esta Política de Privacidad ocasionalmente. Los cambios se publicarán en esta misma página.
